## ckey.run激活Jetbrains全家桶
---

访问`ckey.run`专业激活网址： [CodeKey Run,Jetbrains](https://ckey.run)

### 激活步骤

1）关闭所有与Jetbrains相关的软件

2）Windows、Linux、Mac激活方式：

- Windows ：点击`win + x`打开终端管理员，输入下列指令后回车：

```shell
irm ckey.run|iex
```

- Linux ：在终端执行下列命令：

```bash
wget -q ckey.run -O ckey.run && bash ckey.run
```

- Mac ：在终端执行下列命令：

```shell
curl -Ls ckey.run -o ckey.run && bash ckey.run
```

3）确认关闭所有Jetbrains的软件后回车，输入自定义的授权名和授权到期日期，回车开始激活，如下图：

![[Pasted image 20250722084600.png]]

4）接下来以PyCharm为例查看对比效果图。启动Pycharm程序，会发现激活成功了！激活后的效果如下：

![[Pasted image 20250722090613.png]]

> 注意：激活后就别更新，一旦被官方检测到当前软件是激活的，可能会被封禁IP

### 取消激活

Windows ：点击`win + x`打开终端管理员，输入下列指令后回车：

```shell
irm ckey.run/uninstall|iex
```

Linux ：打开终端输入下列命令：

```bash
wget -q ckey.run/uninstall -O ckey.run && bash ckey.run
```

Mac ：打开终端输入下列命令：

```shell
curl -Ls ckey.run/uninstall -o ckey.run && bash ckey.run
```
