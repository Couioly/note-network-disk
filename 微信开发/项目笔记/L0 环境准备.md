### **第一步：下载官方安装包**

这是**最最重要**的一步，务必从微信官方渠道下载。

- **官方下载地址**：[https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)[](https://hsk.oray.com/news/39018.html)

- **如何选择版本**：
	- 打开页面后，你会看到 **“稳定版 (Stable Build)”** 的下载选项。
	- 根据你的电脑操作系统，选择 **Windows** 或 **macOS** 对应的版本下载即可。如果你的 Mac 是 M1 或更新的苹果芯片，请选择 **macOS ARM64** 版本。
	- **新手提示**：对于刚接触的新手，直接下载并安装“稳定版”是最佳选择。


### **第二步：安装并打开工具**

下载完成后，像安装其他普通软件一样，运行安装包，按照提示完成安装。

- **在Windows上**：安装完成后，通常在桌面会有“微信开发者工具”的快捷图标，双击即可打开。
- **在Mac上**：将应用拖入“应用程序”文件夹，然后在“应用程序”中找到并打开它。
- **登录**：首次打开后，你会看到一个登录界面。**使用你的微信扫描二维码登录**即可。

### **第三步：创建你的第一个小程序项目**

登录成功后，就可以创建项目了。我们先使用“测试号”来体验。

1. 点击左上角的 **“+”** 号或 **“新建”** 按钮。

2. 在弹出的窗口中填写项目信息：
	- **项目名称**：可以自定义，例如 “MyFirstMiniProgram”。
	- **目录**：选择一个空文件夹来存放你的项目代码。
	- **AppID**：这是关键。**点击下拉菜单，选择“测试号”**。这能让你在没有正式申请的情况下进行开发。
	- **开发模式**：选择 **“小程序”**。
	- **后端服务**：为了简化，首次创建时建议选择 **“不使用云服务”**。

3. 点击 **“新建”**，你的第一个小程序项目就创建完成了！

### **📚 核心知识点与概念**

在安装和创建项目时，你会接触到几个核心概念，现在可以简单了解：

| 概念        | 它是什么？          | 新手阶段需要知道的是…                     |
| --------- | -------------- | ------------------------------- |
| **AppID** | 小程序的唯一身份证。     | 开发时可以使用 **“测试号”** 代替，方便学习。      |
| **模拟器**   | 工具主界面左侧的手机屏幕。  | 你写的代码会**实时显示**在这里，模拟手机运行效果。     |
| **编辑器**   | 工具主界面中间的代码区域。  | 你将要在这里编写 WXML、WXSS、JS 等代码文件。    |
| **调试器**   | 工具主界面下方或右侧的面板。 | 查看代码错误、网络请求和存储数据，是**排查问题的控制台**。 |

### **⚠️ 常见安装问题与解决方法**

如果在安装或启动时遇到问题，可以参考下面的方法排查：

| 问题                        | 可能原因与解决方案                                   |
| ------------------------- | ------------------------------------------- |
| 工具**无法启动**（Windows 11常见）  | 可能是权限或兼容性问题。可以尝试以**管理员身份**运行，或参考官方社区的解决方案。  |
| 首次打开**模拟器空白**或**基础库下载失败** | 通常是网络问题。可以检查网络连接，或在工具设置中尝试切换 **“基础库”** 的版本。 |
| **没有 AppID**，无法创建项目       | 无需担心。在创建项目时，AppID 一栏**直接选择“测试号”**即可开始开发。    |
