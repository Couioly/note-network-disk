### 数据采集时防被锁IP

- 创建自己的 `User-Agent` 池；
- 调用 `fake_useragent` 模块。

```python
import fake_useragent 
User_Agent = fake_useragent.UserAgent().random 
headers = {"User-Agent":User_Agent}
```
