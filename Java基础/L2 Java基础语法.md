
## 1 Java程序的基本结构
---
一个典型的Java程序包括**包声明、导入类、定义类、入口方法、主体代码**等多个组成部分，这些部分共同协作，形成**完整的Java程序**。

```java
package cn.itcast.myapp;     // 包声明  
import java.util.Scanner;    // 导入类  
  
public class Main {          // 定义类  
    public static void main(String[] args) { // 入口方法  
        // 主体代码  
        Scanner sc = new Scanner(System.in);  
        System.out.print("请输入您的名字：");  
        String name = sc.nextLine();  
        System.out.println("您好，" + name + "!");  
    }  
}
```

### 1.1 包声明

Java中的包是一种组织和管理类文件的机制，它将相关的类组织到一个包，以便**更好地管理和维护代码**。包可以**嵌套使用**，形成**层级结构**，通常每个包都有一个**唯一的名称**，称为**包名**。包使用的关键字`package`声明，具体语法如下：

```java
package 包名;
```

>[!tips] 
>上述语法格式中，包名用于给类和接口提供**唯一标识符**，避免不同包中同名类产生冲突。包的声明只能位于**Java源文件**的**第一行**。且Java中的包是**可选**的，没有显示声明包时，类被放置在默认包中。

### 1.2 导入类

在Java开发过程中，如果一个**类**要使用**另一个包中的类**，则需要使用`import`关键字来导入这个类，使用import关键字导入类的语法格式：

```java
import 包名.类名;
```

导入类的语句通常出现在**声明包**的语句**之后**，导入类的语句也是**可选**的，使用import关键字导入指定包下的类后，就**不必**在每次使用到该类时，都书写**类的全限定名**。如果需要用到**一个包**中的**多个类**，则可以使用如下语法格式，导入该包下所有的类：

```java
import 包名.*;
```

### 1.3 定义类

Java的所有代码都需要在**类**中书写，类可以被看做是Java程序的**基本单元**。类的定义使用关键字`class`，具体语法格式如下：

```java
修饰符 class 类名{
	//类体
}
```

>[!tips]
>在上述语法格式中，修饰符分为**访问修饰符**和**非访问修饰符**，访问修饰符用于**控制类的访问权限**，例如`public`是**公共访问修饰符**，非访问修饰符有`final`、`abstract`等等。

### 1.4 入口方法

**入口方法**也叫**主方法**。入口方法是**程序执行**的**起始位置**，具体语法格式：

```java
public static void main(String[] args) {
	// 入口方法主体代码
}
```

>[!warning]
>此处只需知道**Java程序执行**时从入口方法开始即可

### 1.5 主体代码

主体代码位于**入口方法**中，用于实现程序的**具体逻辑**。主体代码还可以使用**变量**和**调用方法**来完成**特定任务**。

>[!info] 编写Java
>1. 使用`public`修饰的类名和文件名需要**保持一致**
>2. Java**严格区分大小写**。变量名、方法名、类名等的大小写需要**保持一致**
>3. Java程序中每条语句需要以`;`结束，否则程序会报错
>4. 在编写程序时，为了增强代码的可读性，通常会使用空格或制表符来构建代码的**层次结构**，一般使用**四个空格**作为一个**缩进级别**。此外，在编写程序时通常**一行只写一条语句**，符号`{`与代码同行，符号`}`单独占一行

## 2 Java程序中的注释
---
在Java中，**注释**是用来为代码添加**解释、说明、文档化信息**的特殊文本，它**只在Java源文件中有效**，在**编译程序**时编译器会**忽略**这些注释信息，**不会**将其编译到**字节码文件（.class)中**。

Java中的注释有三种类型，分别是**单行注释、多行注释、文档注释**。

### 2.1 单行注释

用于对程序中的**某一行代码**进行解释。单行注释以符号`//`开头，可以添加到代码的任意位置，如：

```java 
int x = 5;   // 定义一个变量并为其赋值为5
```

### 2.2 多行注释

注释的内容可以为**多行**，常用于对**整个类**或**方法**的**功能**进行说明。多行注释以符号`/*`开头，以符号`*/`结尾，如：

```java
/* ----------
该方法用于输出
Hello World!
公共静态的方法
  ----------*/
public static void printstr(){
	System.out.println("Hello World!");
}
```

>[!warning] 注释的嵌套使用
>1. 多行注释`/* …… */`中**可以**嵌套使用单行注释 `//`
>2. 多行注释`/* …… */`中**不能**嵌套使用多行注释`/* …… */`

### 2.3 文档注释

是为了**生成项目文档**或**API文档**而设计的注释形式，用于对一段代码运行**概括性**的解释说明，文档注释通常被用于描述**类、接口、方法、字段**等代码元素的**用途、输入参数、返回值、异常**等信息。

文档注释以`/**`开始，以`*/`结束，位于代码元素的前面。在文档注释中，可以使用特定的标签，例如`@param`、`@return`、`@throws`等标签提供详细的解释和说明。例如：

```java
/**
	* 入门的Java程序
	* @author Couioly   // 注释此文档代码的作者
	* @version 1.0      // 注释文档代码的版本信息
*/
```

## 3 Javadoc工具
---
`javadoc` 是Java开发工具包JDK中的一个命令行工具，用于从**Java源代码文件生成API文档**。这些生成的文档通常以**HTML**格式呈现，提供了**对类、方法、字段**等的**详细描述**，以及相应的注释。

文档注释可以使用**javadoc**工具生成**HTML API文档**，使用javadoc工具的命令：

```shell
javadoc [options] [source files]
```

**Options**用于设置参数，常用参数：
- `-d <directory>`：指定输出目录，存放生成的HTML文件
- `sourcepath <pathlist>`：指定源文件路径，可以是多个路径，用分隔符`;`或`:`分隔
- `encoding <encoding>`：指定源文件的字符编码
- `charset <charset>`：指定生成文档时使用的字符集

## 4 关键字
---
**关键字**是编程语言中具有**特殊含义**和**功能**的**预定单词**。在Java中，关键字用于**声明变量、定义类、控制程序流程、处理异常、实现面向对象**的特性等，例如`class`关键字用来定义类，`package`关键字用来定义定义包等。

>[!info] Java关键字
>1. 所有关键字都是**小写**的
>2. const和goto是**保留关键字**，虽然在Java的当前版本中还**没有任何意义**，但在程序中**不能**用来作为**自定义的标识符**
>3. true、false和null虽然**不适于关键字**，但它们具有特殊意义，也**不能作为标识符使用**
>4. 自java10开始，Java中增加了**局部变量类型推断功能**，支持使用**var定义变量**。但是**var不属于Java中的关键字**，而是保留类型名，在程序中同样**不能**用作**自定义的标识符**

## 5 标识符
---
Java中的标识符用于**唯一**的标识程序的各种元素，如**变量、方法和类**等，Java中的标识符在定义时，需要遵循如下规则：

- 标识符可以由 **字母、数字、下划线`_`和美元符`$`** 组成，但**不能以数字开头**；
- 标识符是区分大小写；
- 不能是**Java关键字**；
- 以上规则必须遵守，否则将会报错。

>[!tips] 标识符介绍
>为了增强程序中代码的可读性，降低编程人员的沟通成本，提升效率，在定义标识符同时遵循以下规则：
>1. 包含所有字母一律小写；
>2. 类名和接口名每个单词的首字母都要大写；
>3. 常量名所有的字母都要大写；
>4. 变量名和方法名一般使用首字母小写的驼峰命名法（第一个单词首字母小写，从第二个开始每个单词首字母大写）；
>5. 在程序中，应尽量使用有意义的命名标识符；
>6. 标识符的长度没有限制，但不要过长。

### 6.1 数据存储原理

计算机内存由无数个电容器组成，可以通过电路控制给电容充电或放电，充满电代表1，未充电代表0，所以计算机存储任何数据只能用0和1来表示，每个电容就是一个最小存储单位，称为bit（位）。

计算机表示数字只能使用0或1，当数字大于1时，就必须进位，也就是**逢2进1**，称为**二进制**。
每4个二进制作为1位数，范围是0~15，大于9的用A~F代替，每**逢16进1**，称之为**十六进制**。
每3个二进制位作为1位数，**逢8进1**，称为**八进制**。

```java
public class Main{
	public static void main(String[] args){
		// 十进制的10
		System.out.println(10);
		// 二进制的10，以0b开始
		System.out.println(0b1010);
		// 八进制的10，以0开始
		System.out.println(012);
		// 十六进制的10，以0x开始
		System.out.println(0xA);
	}
}
```

### 6.2 字符存储原理

**ASCII码**：美国信息交换标准编码，规定了数字字符、英文字母、其他常用符号等共计128个字符及对应的数值编码，每个字符对应的数值称为**码点**。每8个bit称为一个Byte（字节），简写B。（B->KB->MB->GB->TB）

**GBK码**：全称《汉字内码扩展规范》，兼容ASCII码表的同时，扩展了21003个汉字，用两个字节来表示字符。

**Unicode码**：也称**统一码**或万国码，是由ISO（国际标准化组织）制定的码表。包含世界上大部分国家的大多数文字和字符，大约14万多。该编码表的实现方式：

- UTF-8：变长的编码方式，它可以使用1~4个字节表示一个符号，根据不同的符号而变化字节长度。
- UTF-16：变长的编码方式，基础字符用2个字节，补充字符用4字节。
- UTF-32：固定长度编码，不管字符的Unicode码多大，统一使用4个字节。

### 6.3 图像和音频存储

计算机中的图片都是由**像素点**组成的，每个像素点有各自的颜色，当像素点足够小，足够多时，人眼无法识别，就变成了一幅画。每个像素点都包含红R、绿G、蓝B三个子像素，称为三基色。每个颜色的亮度都用0~255的数值来表示，数值越大越亮，反之越暗。

计算机存储声音通常会经过三个步骤：采样、量化、编码

## 8 变量
---
在程序运行期间，随时可能产生一些临时数据，应用程序会将这些**数据保存在内存单元**中，每个内存单元都用一个**表示符**标识，这些内存单元被称为**变量**，用于**标识内存单元的标识符**就被称为**变量名**，内存单元中**存储的数据**就是**变量的值**。

### 8.1 变量的声明

**变量的声明**是指在代码中明确指定**变量的类型**和**名称**，以便编译器**识别变量**并为其**分配合适的存空间**。声明语法格式如下：

```java
数据类型 变量名[,变量名2，……];
```

- 变量名是为变量指定的标识符，用于**唯一的标识变量**；
- 变量的命名需要遵守**标识符的命名规则**；
- 一条语句中**可以同时声明多个变量**，变量名之间用**逗号分隔**。

### 8.2 变量的定义

变量的定义是指在声明的基础上为**变量名分配内存空间**并选择性的**为其赋予初始值**，可以**在声明变量时进行赋值操作**，也可以在**后续代码中进行赋值操作**，定义变量的语法格式如下：

```java
数据类型 变量名 [=变量值][,变量名2[=变量值]……];
```

变量具有一定的**可访问范围**，这个范围被称为**变量的作用域**。

>[!warning] 使用逻辑运算符的注意事项
>运算符`&`、`|`在做逻辑运算时，与`&&`和`||`具有**相似的功能**，但是它们在使用上由一些**关键区别**：
>- 在使用`&`进行运算时，不论左边表达式的值为**true还是false**，右边的表达式**都会被执行**，而在使用`&&`进行运算时，若左边表达式值为false，右边的表达式将**不会被执行**；
>- 在使用`|`进行运算时，不论左边的值为**true还是flase**，右边的表达式**都会被执行**。而在使用`||`进行运算时，如果左边的值为true，右边的表达式将**不被执行**。

**示例：**

```java
public class Main {          // 定义类  
    public static void main(String[] args) { // 入口方法  
        int x=0;  
        int y=0;  
        int z=0;  
        boolean a,b;  
        a = x > 0 & y++ > 1;  
        System.out.println(a);  
        System.out.println("y = "+y);  
        b = x > 0 && z++ > 1;  
        System.out.println(b);  
        System.out.println("z = "+z);  
    }  
}
```

运行结果为：

```
false
y = 1
false
z = 0
```

>[!tips] 三元运算符
>三元运算符又称条件运算符，用于根据条件的真假来选择返回不同的值，三元运算符的语法格式：
>>条件表达式 ? 表达式1 : 表达式2;
>
>>[!warning] 注意事项
>>1. 三元运算符的优先级低于比较运算符与算数运算符，但高于赋值运算符；
>>2. 三元运算符的最终结果一定要被使用，或者赋值给一个变量，或者直接输出；
>>3. 三元运算符可以进行嵌套，运算方向自右向左，例如`a>b?a:c>d?d:d`应该理解为`a>b?a:(c>d?c:d)。

## 9 数据类型的转换
---
Java中的数据类型的转换形式分为两种：
- 自动类型转换
- 强制类型转换

### 9.1 自动类型转换

**自动类型的转换**又称为**隐式类型转换**，指的是两种数据类型在转换的过程中不需要显示地进行声明，由**编译器自动完成**。自动类型地转换需要同时满足两个条件：

- **两种数据类型彼此兼容**
- **目标类型的取值范围大于源类型的取值范围**

**可自动类型转换的情况**：
1. 整数类型之间可以实现转换，以下几种可以实现转换：
	- byte类型的数据可以赋值给short、int、long类型的变量；
	- short、char类型的数据可以赋给int、long类型的变量；
	- int类的数据可以赋给long类型的变量
2. 整数类型转换为float类型，如byte、char、short、int类型的数据可以赋给float类型的变量
3. 其他类型转换为double类型，如byte、char、int、short、long、float类型的数据可以赋值给double类型的变量

### 9.2 强制类型转换

**强制类型转换**又称为**显式类型转换**，指的是两种数据类型之间的转换需要进行**显示的声明**，当两种类型**彼此不兼容**，或者**目标类型**取值范围**小于源类型**时，自动类型转换无法进行，这时需要强制类型转换。强制类型转换语法格式:

```java
目标数据类型 变量名 = (目标类型) 值;
```

强制类型转换时，如果将**取值范围较大**的数据类型**强制转换**为**取值范围较小**的数据，可能会导致**数据的截断**或**精度丢失**。

## 10 控制台输入数据
---
在Java程序中，如果想要读取用户输入到命令行的内容，可以使用`Scanner`类，用于读取用户输入和文件等数据源，Scanner类可以扫描并接收控制台输入的各种类型的数据，包括整数、小数、字符串等。

使用Scanner类接收从控制台输入数据的方法步骤如下：

```java
import java.util.Scanner;
```

2）创建Scanner对象

```java
Scanner sc = new Scanner(System.in);
```

3）调用Scanner类的相关方法接收不同类型的数据

```java
int num = sc.nextInt();
String str = sc.next();
float f = sc.sc.nextFloat();

```

## 11 选择结构语句
---
Java程序中，选择结构语句是一种流程控制语句，它可以根据特定的条件来选择性的执行不同的语句块，java中常见的选择结构有if语句和switch语句。

if语句用于根据某个条件的结果决定是否执行特定的代码块，if语句有三种应用格式，分别为if语句，if...else语句和if...else if...else语句。

if语句的语法与C语言类似，这里不再赘述。

### switch语句

switch语句通过某个表达式的结果或某个变量的值做出判断，从而决定程序执行哪一段代码。其语法格式如下：

```java
switch (表达式或变量){
	case 目标值1:
		语句或代码块1
		break;
	case 目标值2:
		语句或代码块2
		break;
	...
	case 目标值n:
		语句或代码块n
		break;
	default:
		语句或代码块n+1
		break;
}
```

将表达式或变量的值与每个case后的目标值进行匹配，找到匹配的值，则执行对应的case语句，若未找到，则执行default的语句。

>[!warning] 注意事项
>- switch语句中的表达式支持的数据类型只有整数类型、枚举类型、字符类型、字符串类型，不支持浮点数类型和布尔类型；
>- case条件后面的目标值不允许重复；
>- 正常使用switch时，case后面的语句中最后一行需要使用break语句，否则会发生“穿透”现象；
>- 在使用switch语句时，若多个case条件后面的执行语句一样，则执行语句只需要书写一次即可。

## 12 循环语句
---
**while循环**的语法格式如下：

```java
while (循环条件){
	循环体
}
```

**do...while循环**的循环条件放置在后面，这意味着循环体会无条件执行一次，然后进行判断，其语法格式如下：

```java
do {
	循环体
} while (循环条件);
```

**for循环**一般用在**循环次数已知**的情况下，for循环的语法格式：

```java
for(初始化表达式;循环条件;操作表达式){
	语句或代码块
}
```

两个跳转语句介绍：
- `break`：终止并跳出循环语句（switch语句）
- `continue`：终止本次循环，进入下一次循环

## 13 数组
---
变量可以用来存储和操作程序运行过程中的数据，但基本类型的变量只能存储单个数据，数组是一个容器，可以用来存储同种数据类型的多个值，每个值称为数组的元素。

### 13.1 数组的声明

数组声明的格式：

```java
// 方式一
type[] arrayName;
// 方式二
type arrayName[];
```

- `type`：数组元素的数据类型，可以是基本数据类型，也可以是引用数据类型
- `arrayName`：数组名称

### 13.2 数组的初始化

Java语言中数据必须初始化后才可以使用，所谓初始化，就是**为数组元素分配内存空间**，并**为每个数组元素赋初始值**，Java中数组的初始化有静态初始化和动态初始化两种方式。

1）**静态初始化**
静态初始化是指数组初始化时，**显示指定每个数组元素的初始值**，静态初始化的语法格式如下：

```java
// 格式一
type[] arrayName = new type[](元素1,元素2,...);
// 格式二
type[] arrayName = {元素1,元素2,...};
```

**2）动态初始化**
动态初始化是指数组初始化时**只指定数组长度**，即指定数组中存放数组元素的个数，由系统为数组元素分配初始值。语法格式如下：

```java
type[] arrayName = new type[length];
```

- `length`：指数组的长度，也就是数组可容纳的元素个数，指定数组的长度后，数组的长度就不能再修改

动态初始化时，系统为数组元素分配初始默认值，根据元素类型的不同，默认初始值也不同：

|        数据类型         | 默认初始值            |
| :-----------------: | :--------------- |
| byte、short、int、long | 0                |
|    float、double     | 0.0              |
|        char         | 一个空字符，即'\u0000'  |
|       boolean       | false            |
|       引用数据类型        | null，表示变量不引用任何对象 |
当数组的长度确定后可以使用`arrayName.length`获取数组的长度。
可以通过数组的索引来访问数组元素，格式`数组名[索引]`，数组的索引值从0开始，到数组长度-1结束。

### 13.3 二维数组

二维数组就是指**维度为2的数组**，即数组由**两个索引**，二维数组的逻辑结构按行列排列，两个索引分别表示行和列，通过行和列可以准确标识一个元素。二维数组的定义方式：

```java
// 方式1：确定行数和列数
数据类型[][] 数组名 = new 数据类型[行数][列数];
int[][] arr = new int[3][4];
// 方式2：确定行数不确定列数
数据类型[][] 数组名 = new 数据类型[行数][];
int[][] arr = new int[3][];
// 方式3：确定元素值
数据类型[][] 数组名 = {{第0行初始值},{第1行初始值},...{第n行初始值}};
int[][] arr = {{1,2},{3,4,5,6},{7,8,9}};
```

二维数组中元素的访问也通过索引方式：`数组名[行索引][列索引]`。

## 14 方法
---
Java中方法是用来**执行特定任务**的一组可重复调用的语句集合，将会被重复执行的代码提取出来，封装在一个方法内，并为其命名，开发者可以在需要执行这些代码的位置直接调用该方法，避免了重复编写形同的代码。

方法的定义是指编写方法的具体格式和逻辑，以便程序能够正确地识别并执行方法，方法地定义格式：

```java
[修饰符] 返回值类型 方法名([参数列表]){
	// 方法体，即方法要执行的代码
	return 返回值; // 用于返回方法执行结果
}
```

- **修饰符**：可选项，用于控制方法或类的访问权限
- **返回值类型**：用于指定方法执行完毕后返回的数据类型，无返回值时使用void
- **方法名**：指定方法名称的标识符
- **参数列表**：可选项，用于定义方法接收的参数，参数之间使用逗号间隔
- **return**：返回方法指定类型的值，若返回值类型为void，则不需要写return
- **返回值**：被return返回的值，该值会返回给方法调用者

**方法的重载**

方法的重载是指在**同一个类**中可以定义**多个同名**但**参数列表不同**的方法。重载方法的参数列表可以通过**参数数量、参数类型、参数顺序**来区分，需要注意的是，方法的重载与返回值类型无关。

**示例**：

```java
public static int add(int x,int y){
	return x+y;
}
public static int add(int x,int y,int z){
	return x+y+z;
}
public static double add(double x,double y){
	return x+y;
}
```

调用add()方法：

```java
int sum1 = add(1,2);
int sum2 = add(1,2,3);
double sum3 = add(1.2+1.1);
```
