
## 一、Obsidian 特有引用语法（核心重点）

Obsidian 的引用功能围绕 “双向链接”“块引用”“嵌入引用” 设计，支持精准定位笔记内容，以下是完整语法及用法：

### 1. 笔记引用（双向链接）

* **基础语法**：`[[笔记名称]]`
  * 示例：`[[2024年工作计划]]` → 生成指向 “2024 年工作计划” 笔记的双向链接

* **带别名引用**：`[[笔记名称|显示别名]]`
  * 示例：`[[2024年工作计划|今年规划]]` → 显示 “今年规划”，点击跳转目标笔记

* **跨文件夹引用**：`[[文件夹名称/笔记名称]]`
  * 示例：`[[项目文档/产品需求说明书]]` → 引用子文件夹中的笔记

* **模糊匹配引用**：`[[部分笔记名称]]`（Obsidian 自动联想匹配）
  * 示例：`[[工作]]` → 自动弹出包含 “工作” 关键词的所有笔记供选择

### 2. 块引用（精准引用笔记内特定内容）

块引用需先选中目标内容生成 “块 ID”（Obsidian 自动分配，右键可复制块 ID），支持段落、列表、代码块等任意内容块。

* **基础语法**：`[[笔记名称#^块ID]]`
  * 示例：`[[产品需求说明书#^3f7d92]]` → 直接跳转到该笔记中 ID 为 3f7d92 的段落

* **带别名块引用**：`[[笔记名称#^块ID|显示别名]]`
  * 示例：`[[产品需求说明书#^3f7d92|用户登录流程说明]]`

* **同笔记内块引用**：`[[#^块ID]]`（无需写笔记名称）
  * 示例：`[[#^a1b2c3]]` → 引用当前笔记中 ID 为 a1b2c3 的内容块

### 3. 标题引用（跳转到笔记内指定标题）

* **基础语法**：`[[笔记名称#标题名称]]`
  * 示例：`[[产品需求说明书#用户注册流程]]` → 跳转到目标笔记的 “用户注册流程” 标题下

* **多级标题引用**：`[[笔记名称#一级标题#二级标题]]`
  * 示例：`[[产品需求说明书#登录功能#第三方登录]]` → 跳转到多级子标题

* **同笔记内标题引用**：`[[#标题名称]]`
  * 示例：`[[#核心功能清单]]` → 跳转到当前笔记的对应标题

### 4. 嵌入引用（直接显示目标内容，而非跳转链接）

在普通引用语法前加`!`，可将目标笔记、块、标题内容嵌入当前笔记。

* **嵌入笔记**：`![[笔记名称]]`
  * 示例：`![[每周会议纪要]]` → 在当前笔记中显示 “每周会议纪要” 的完整内容

* **嵌入块**：`![[笔记名称#^块ID]]` 或 `![[#^块ID]]`（同笔记内）
  * 示例：`![[产品需求说明书#^789abc]]` → 嵌入目标块的内容（如一段文字、一个表格）

* **嵌入标题**：`![[笔记名称#标题名称]]` 或 `![[#标题名称]]`（同笔记内）
  * 示例：`![[#数据统计图表]]` → 嵌入当前笔记中 “数据统计图表” 标题下的所有内容

* **嵌入图片（Obsidian 兼容 Markdown 图片语法，补充说明）**：`![图片描述](图片路径)`
  * 示例 1（本地图片）：`![截图](attachments/20240501_screenshot.png)`（推荐放在 attachments 文件夹统一管理）
  * 示例 2（网络图片）：`![Obsidian图标](https://obsidian.md/images/obsidian-icon.svg)`

### 5. 标签引用（关联笔记标签）

标签本质是分类标记，可通过标签面板筛选笔记，语法与引用联动：

* **标签语法**：`#标签名称`
  * 示例：`#工作 #项目A #待办`

* **多级标签**：`#一级标签/二级标签`
  * 示例：`#工作/项目A/设计任务`

* **标签引用（筛选跳转）**：点击笔记中的`#标签名称`，直接跳转到标签面板显示所有带该标签的笔记

## 二、常用 Markdown 基础语法（Obsidian 完全兼容）

以下是日常笔记高频使用的 Markdown 语法，与 Obsidian 引用功能配合使用更高效：

### 1. 标题

* 语法：`# 一级标题`（1-6 个`#`对应 1-6 级标题）

```
\# 一级标题（最大）
\## 二级标题
\### 三级标题
\#### 四级标题
\##### 五级标题
\###### 六级标题（最小）
```

### 2. 文本格式

* 加粗：`**加粗内容**` 或 `__加粗内容__` → **加粗内容**
* 斜体：`*斜体内容*` 或 `_斜体内容_` → *斜体内容*
* 粗斜体：`***粗斜体内容***` → *粗斜体内容*
* 删除线：`~~删除线内容~~` → ~~删除线内容~~
* 高亮（Obsidian 支持）：`==高亮内容==` → == 高亮内容 ==
* 代码行：`代码内容`（反引号包裹）→ `console.log("Hello")`

### 3. 列表

* 无序列表：`- 列表项` 或 `* 列表项` 或 `+ 列表项`（可嵌套）

```
\- 项目1
&#x20; \- 子项目1-1
&#x20; \- 子项目1-2
\* 项目2
\+ 项目3
```


* 有序列表：`数字. 列表项`（自动排序，可嵌套）


```
1\. 第一步
&#x20; 1\. 子步骤1
&#x20; 2\. 子步骤2
2\. 第二步
3\. 第三步
```


* 任务列表：`- [ ] 未完成任务` / `- [x] 已完成任务`

```
\- \[x] 整理语法文档
\- \[ ] 测试Obsidian引用功能
\- \[ ] 补充示例笔记
```

### 4. 链接与图片（补充完整语法）

* 外部链接：`[链接文本](链接地址 "可选描述")`
  * 示例：`[Obsidian官网](https://obsidian.md/ "Obsidian Official Site")`

* 锚点链接（同笔记内）：`[跳转到标题](#标题名称)`
  * 示例：`[跳转到引用语法](#一、Obsidian-特有引用语法（核心重点）)`

* 图片（含尺寸调整，Obsidian 支持）：`![描述](路径 =宽度x高度)`
  * 示例：`![截图](attachments/screenshot.png =800x600)`

### 5. 代码块

* 语法：`语言名称 + 代码 + `（指定语言可实现语法高亮）

```
def hello\_obsidian():

&#x20;   print("Hello, Obsidian!")

hello\_obsidian()
```

* 无语法高亮：`+ 代码 +`（不指定语言）

### 6. 表格

* 语法：`| 表头1 | 表头2 | 表头3 |` + 分隔行 + 内容行

```
\| 语法类型       | 核心用途               | 示例语法                     |
\|----------------|------------------------|------------------------------|
\| 笔记引用       | 双向链接笔记           | \`\[\[笔记名称]]\`               |
\| 块引用         | 精准引用内容块         | \`\[\[笔记名称#^块ID]]\`         |
\| 嵌入引用       | 直接显示目标内容       | \`!\[\[笔记名称#标题]]\`         |
```

* 对齐方式（可选）：分隔行添加`:`，如`| :--- | :---: | ---: |`（左对齐、居中、右对齐）

### 7. 引用块（Markdown 原生，区别于 Obsidian 块引用）

* 语法：`> 引用内容`（可嵌套，加多个`>`）

```
\> 这是一级引用
\>> 这是嵌套的二级引用
\>>> 这是三级引用
```

### 8. 分割线

* 语法：`---` 或 `***` 或 `___`（单独一行，前后空行）

```
内容上方
\---
内容下方
```

### 9. 脚注

* 语法：`正文内容[^脚注标识]` + 文末`[^脚注标识]: 脚注内容`

```
Obsidian 是一款强大的笔记工具\[^1]
\[^1]: 基于 Markdown 语法，支持双向链接、块引用等核心功能。
```

### 10. 数学公式（Obsidian 需开启数学公式功能）

* 行内公式：`$公式内容$`        示例：`$E=mc^2$` → $E=mc^2$
* 块级公式：`$$公式内容$$`（单独一行）

```
\$\$
a^2 + b^2 = c^2
\$\$
```
